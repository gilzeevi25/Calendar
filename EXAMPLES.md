# דוגמאות שימוש בבוט

## יציאה (Leave)
```
גיל זאבי יוצא הביתה ב-15.05 וחוזר ב-20.05
```

## יציאה עם לוח שלם (Leave + full calendar)
```
גיל זאבי במוצב מ-01.05 עד 30.05, יוצא ב-15.05 וחוזר ב-20.05
```

## ביטול חופשה (Cancel Leave)
```
גיל זאבי מבטל את החופשה מ-15.05 עד 20.05
```

## הארכת חופשה (Extend Leave)
```
גיל זאבי מאריך עד 22.05
```

## הוספת אדם (Add Person)
```
תוסיף את גיל זאבי
```

## הסרת אדם (Remove Person)
```
תסיר את גיל זאבי
```

## הערה (Add Note)
```
הערה לגיל זאבי ב-15.05: תורנות
```

## החלפה (Swap)
```
גיל זאבי ויוסי כהן מחליפים מ-15.05 עד 20.05
```

## עדכון סטטוס (Bulk Status)
```
כולם בפעילות מ-01.05 עד 10.05
```

## תוכנית מחזורית (Schedule)
```
תוכנית יציאות לגיל זאבי מתכונת 10-4 מ-01.05 עד 30.06
```

## העתקת הערות (Copy Notes)
```
תעתיק הערות מגיל זאבי ליוסי כהן
```

## העתקת תוכנית (Copy Plan)
```
תעתיק תוכנית מגיל זאבי ליוסי כהן
```

## שאילתה (Query)
```
מי בבית ב-15.05?
```
```
מה הסטטוס של גיל זאבי?
```

## ביטול שינויים (Undo)
```
תבטל שינויים
```
