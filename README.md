# לוח פעילויות — Hebrew Interactive Calendar

לוח שנה אינטראקטיבי בעברית להצגת לוחות זמנים אישיים (פעילות / בבית) לתאריכים 26.4.2026 – 18.6.2026.

## הפעלה

```bash
cd Calendar
python -m http.server 8000
```

פתחו בדפדפן: [http://localhost:8000](http://localhost:8000)

## עריכת נתונים

ערכו את הקובץ `data.csv` בפורמט הבא:

```csv
name,date,status
אדם 1,2026-04-26,activity
אדם 1,2026-04-27,home
```

- **name** — שם האדם (עברית)
- **date** — תאריך בפורמט `YYYY-MM-DD`
- **status** — `activity` (פעילות) או `home` (בבית)

## תכונות

- תצוגת לוח שנה עברי (RTL) עם תאריכים עבריים
- סימון שבתות ושבועות
- סינון לפי אדם או תצוגת סיכום לכולם
- עיצוב רספונסיבי ותמיכה בהדפסה
